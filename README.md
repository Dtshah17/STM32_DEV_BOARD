## Overview

The STM32 Dev Board is a simple PoC design utilising the STM32F070C6 MCU

## Features 

The devboard features the following:

- USB-C Power and Program Input
- 5V to 3.3V Voltage Regulation 
- 4MB QSPI FLASH Memory 
- External 8MHz Crystal 
- MOSFET Driven BOOT and Reset Switch 
- USART Interface
- SWD Debugging Interface
- Host I2C Protocal Interface
- Status LEDs
